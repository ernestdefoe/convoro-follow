# Follow & Feed (convoro-follow)

A first-party Convoro extension that lets members **follow each other** and get a
**personal feed**.

A Follow button appears on member profiles (with a follower count); the latest
topics and replies from people you follow collect on a `/feed` page (linked from
the header when you're signed in). Opt-in — install it only if your community
wants it.

Adds a `follows` table (removed on uninstall). MIT licensed.
